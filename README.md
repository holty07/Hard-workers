# Hard Workers

A Minecraft NeoForge 1.21.1 mod that adds autonomous worker blocks to automate resource gathering and farming.

## Workers

Four worker types are available, each in five tiers (Wood, Stone, Iron, Diamond, Netherite):

- **Lumberjack** — finds and chops trees, then replants saplings
- **Miner** — digs a tunnel in the direction the block faces
- **Farmer** — harvests and replants crops in a surrounding area
- **Warehouse Worker** — collects items from other worker blocks and stores them centrally

Higher tiers work faster and (for the Miner) can mine harder blocks.

## Storage

Each worker block has a built-in 27-slot inventory. Workers stop when full. The Warehouse Worker's inventory can be opened by right-clicking the block. All inventories are hopper and pipe compatible.

## Current Version

**v0.1** — NeoForge 21.1.172 / Minecraft 1.21.1

## Installation

Drop the JAR from the [Releases](../../releases) page into your `mods/` folder alongside NeoForge 21.1.172.
